# MP3Browser 
this is updated for python 3 use and has been extracted from the original Kashmir python 2 plugin

Description: MP3 Browser Plugin


Maintainer: TwolDe2(python 3)
Homepage: https://github.com/TwolDE2/MP3Browser
Depends: python3-mutagen
Original Source: python 2 --> http://www.kashmir-plugins.de

<--> Go to first letter in list


1 All MP3s
2 Screen Saver on/off
3 Favorites
4 
5 Shuffle MP3s
6 Folder select
7 Artist select
8 Album select
9 Genre select
0 End of list 


up = scroll up one song in the song list
down = scroll down one song in the song list
right = scroll one page down in the song list
left = scroll one page up in the song list
ok = select item (play song)
stop = stop song
exit = exit MP3Browser
